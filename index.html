Chrome DevTools

Chrome DevTools é um conjunto de ferramentas para desenvolvimento web contruídas diretamente no navegador Google Chrome. DevTools pode ajudar a editar páginas dinamicamente e diagnosticar problemas, ajudando a construir sites melhores mais rapidamente.

Abrir Chrome DevTools

Para abrir o Chrome DevTools, basta apertar o botão direito do mouse sobre o navegador e escolher Inspect.

Inspecionar elemento

Selecionar um elemento para inspecionar.

Device Mode

Escolher o tamanho da tela.

Elements

Fornece ferramentas para manipulação e inspecção do HTML e CSS da página.

Console

Exibe mensagens e permite executar código Javascript.

Sources

Fornece ferramentas para visualizar, editar e depurar o código do documento.

Permite criar breakpoints.

Network

Fornece ferramentas para visualizar e depurar a atividade na rede.

Performance

Fornece ferramentas para visualização do desempenho.

Memory

Fornece ferramentas para corrigir problemas de memória, como:

  • Memory leaks;
  • Memory bloat;
  • Garbage Collector sendo executado frequentemente.

Application

Fornece ferramentas para inspecionar todas as fontes da aplicação.

Security

Fornece ferramentas para inspecionar a segurança de uma página.
